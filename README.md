# 🌅 Gulf of Florida Sunset Loops

Beautiful relaxing beach sunset video optimized for **YouTube**, sleep, study, and video editors.

## 📦 Available Versions

- **6 Hour Loop** → `make_6hr_loop.sh`
- **8 Hour Loop** → `make_8hr_loop.sh`  
- **10 Hour Loop** → `make_10hr_loop.sh` (main)

## Quick Start (No Cloning Needed)

1. Download your original ~29s video
2. Download the script you want from this repo
3. Run it:

```bash
chmod +x make_6hr_loop.sh
./make_6hr_loop.sh your_video.mp4 gulf_sunset_6hr.mp4
```

Same for 8hr and 10hr scripts.
